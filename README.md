# ERU Kayak & Snowboard Kulübü (Snowriders)

Bu proje, Erciyes Üniversitesi Kayak ve Snowboard Kulübü (Snowriders) için geliştirilmiş bir etkinlik kayıt ve yönetim uygulamasıdır.

## Özellikler

- **Modern Tasarım:** Nordic Peak estetiğine sahip, modern ve kullanıcı dostu arayüz.
- **Etkinlik Takibi:** Güncel kayak ve snowboard etkinliklerinin listelenmesi ve detayları.
- **Hızlı Kayıt:** Kullanıcıların etkinliklere kolayca kayıt olabilmesi.
- **Admin Paneli:** Etkinliklerin yönetilmesi ve katılımcı listelerinin takibi.

## Teknolojiler

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express (Render üzerinde barındırılmaktadır)
- **Barındırma:** Firebase Hosting

## Geliştirme

Projeyi yerelde çalıştırmak için:

1. Bağımlılıkları yükleyin: `npm install`
2. Geliştirme sunucusunu başlatın: `npm run dev`
